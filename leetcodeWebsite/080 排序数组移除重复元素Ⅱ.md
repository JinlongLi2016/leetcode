# 080 排序数组移除重复元素Ⅱ



* My Solution: 依次对每个元素计数，如果大于2则只放置两个

```python
class Solution:
    def removeDuplicates(self, nums: List[int]) -> int:  
        k = i = j = 0
        while j < len(nums):
            cnt = 0
            i = j
            while j < len(nums) and nums[i] == nums[j]:
                cnt += 1
                j += 1
            cnt = min(2, cnt)
            p = nums[i]
            for _ in range(cnt):
                nums[k] = p
                k += 1
        return k
```

def removeDuplicates(self, nums):
    i = 0
    for n in nums:
        if i < 2 or n > nums[i-2]:
            nums[i] = n
            i += 1
    return i
