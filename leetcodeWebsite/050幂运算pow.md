_recursive_


```python
class Solution:
    def myPow(self, x: float, n: int) -> float:
        symbol = n > 0
        n = n if n > 0 else -n
        if n == 0 and x != 0:
            return 1
        elif n == 1:
            return x if symbol==True else 1./x
        else:
            t = self.myPow(x, n//2)
            t = t * t * (x if n&1 else 1)
            return t if symbol==True else 1. / t
```

```python
class Solution:
    def myPow(self, x, n):
        if not n:
            return 1
        if n < 0:
            return 1 / self.myPow(x, -n)
        if n % 2:
            return x * self.myPow(x, n-1)
        return self.myPow(x*x, n/2)
```

_Iterative:_

```python
class Solution:
    def myPow(self, x, n):
        if n < 0:
            x = 1 / x
            n = -n
        pow = 1
        while n:
            if n & 1:
                pow *= x
            x *= x
            n >>= 1
        return pow
```
解析：对于此迭代的解法，从算式分解中可以溯源
